# dotfiles

設定ファイルの管理

## ファイル構成
```sh
. 
├── .bash_profile
├── .bashrc
├── README.md 
├── init.sh 
└── install.sh 
```

## 使い方
```sh
$ mkdir dirname && cd dirname
$ git clone https://github.com/Fukkatsuso/dotfiles.git
$ cd dotfiles
$ sh init.sh
$ sh install.sh
```
